# 📈 AI Stock Assistant

An **interactive AI-powered stock analysis dashboard** built with **Streamlit** that helps users analyze stocks using technical indicators and an integrated chatbot for trading insights.

The application allows users to explore stock data, visualize technical indicators like **RSI** and **Moving Averages**, and interact with an **AI assistant** to understand trading signals.

---

## 🚀 Features

* 📊 **Stock Data Visualization**

  * Fetch and display stock market data
  * Interactive charts for price movements

* 📉 **Technical Indicators**

  * RSI (Relative Strength Index)
  * SMA (Simple Moving Average)

* 🤖 **AI Chatbot Assistant**

  * Explains trading indicators
  * Answers questions about RSI, SMA, buy/sell signals
  * Helps beginners understand stock analysis

* 🎨 **Modern Dashboard UI**

  * Custom gradient background
  * Glass-style UI components
  * Clean and interactive Streamlit interface

## 📊 Technical Indicators Explained

### RSI (Relative Strength Index)

* **RSI > 70** → Stock may be **overbought**
* **RSI < 30** → Stock may be **oversold**
* **RSI 30–70** → Neutral trend

### SMA (Simple Moving Average)

* Short SMA crossing **above** Long SMA → **Buy signal**
* Short SMA crossing **below** Long SMA → **Sell signal**
